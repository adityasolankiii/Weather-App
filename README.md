# Weather App ☁️☀️🌧️

A simple and intuitive Weather App built with **HTML**, **CSS**, and **JavaScript** that allows users to get real-time weather information for any city around the world. The app provides current weather conditions such as temperature, humidity, wind speed, and weather description.

## Features

- **Real-time Weather Data:** Fetches live weather information using a weather API.
- **Search Functionality:** Allows users to search for any city and view its weather details.
- **Responsive Design:** Fully responsive layout for a seamless experience on desktops, tablets, and mobile devices.
- **Weather Icons:** Displays dynamic weather icons that change according to the weather conditions (sunny, cloudy, rainy, etc.).
- **Background Changes:** The app background changes based on the current weather, enhancing the user experience.

## Tech Stack

- **HTML:** For structuring the app layout.
- **CSS:** For styling the user interface and making the app responsive.
- **JavaScript:** For fetching and displaying weather data.

## How to Use

1. **Clone the repository:**

   ```bash
   git clone https://github.com/adityasolankiii/Weather-App.git
   ```

2. **Open the project folder:**

   ```bash
   cd weather-app
   ```

3. **Launch the app:** Open the `index.html` file in a web browser.

4. **Search for a city:** Enter a city name in the search bar to get the current weather details.

## API Used

- [OpenWeatherMap API](https://openweathermap.org/api): Fetches weather data for any city in real-time.
